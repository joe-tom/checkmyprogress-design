Educators
Checkmyprogress has been designed to be easy to use, powerful, and lightweight.There is no software needed to be installed and anyone with an internet connection can use the powerful, web based program.

Checkmyprogress's focus is to provide high quality, easy to use marking software that is simple to use and access.
Improves Communication
- Instant and continuous feedback
- Access to parent/student emails
- Greatly increases school website profile among students and parents
Advantages for Teachers
- Supports traditional and performance based assessment
- Email interims (individually or as a class) at the push of a button
- Email personalized messages home for an individual student or an entire class
- Powerful PDF report generation
- Flexible weighting system
- Students/parents aware sooner of missing work and academic performance
- Easy to use - no technical knowledge required to use
- Secure website
- Data backed up regularly
- Flexible to accommodate unique school and district needs
